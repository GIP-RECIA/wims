互动式 OEF 编辑器功能介绍 (用于创建练习 createxo 的源码模式) :

水平菜单 :

水平菜单中含有生成网页命令 (HTML) 的按钮、插入 WIMS 常用命令的按钮、以及后面的帮助按钮.

第一组按钮用于插入 HTML 的标签. 例如先选取一个词, 然后点击 I 按钮, 你选中的词变成如下的形式 :

<em>选中的词</em>.

将来这个词在你的练习里就以斜体的形式出现.

其它按钮是 OEF 语言特有的, 被分成三组 : "前置", "中置", 及 "后置" :

  • 前置 - 在显示练习以前执行的命令. 主要是练习的源数据 (标题、作者...)
  • 中置 - 用于控制如何在屏幕上显示此练习
  • AFTER - 用于分析用户的回答.

命令表

这里是所有 Wims 命令的表instructions

你可以用 3 中形式插入命令 :

  • 插入 - 只要双击此命令, 或者用单击来选取, 然后再点击下方的 插入选择 按钮.
  • 文件 - 单击一个命令, 就会在文字区域里显示此命令的帮助窗口.
  • 自动完成 - 当你输入一个反斜杠 (\) 以及一个小写字母时, 命令表会自动过滤出与你的字母匹配的命令.

命令 描述
ABOVE tooltip 放在鼠标点上面. 取值: true 或 false.
 
与 OFFSETY 联用可调节到鼠标点的垂直距离, 或用 CENTERMOUSE 使得 tooltip 的中心位于鼠标点上面.
BGCOLOR tooltip 的背景色. 取值: 放在单引号里面的 HTML 色彩, 例如 '#D3E3F6' 或 'DarkCyan', 空色彩 '' 表示 无背景.
例:
onmouseover="Tip('某些文字', BGCOLOR, '#D3E3F6')"
or
onmouseover="Tip('某些文字', BGCOLOR, '')"
BGIMG 背景图像. 取值: 用单引号括起来的到图像的路径.
例:
onmouseover="Tip('某些文字', BGIMG, '../images/tipbackground.gif')"
BORDERCOLOR 边框颜色. 取值: 用单引号括起来的 HTML 色彩, 例如 '#dd00aa'.
BORDERSTYLE 边框格式. 取值: 用单引号括起来的 CSS 边框格式. 推荐值为 'solid' (默认), 'dashed' 或 'dotted', 其它值可能不适用于某些浏览器.
BORDERWIDTH tooltip 边框宽度. 取值: 非负整数. 默认是 1. 0 表示无框.
例:
onmouseover="Tip('某些文字', BORDERWIDTH, 2)"
CENTERMOUSE 使 tooltip 居中处于鼠标下方(或上方). 取值: true 或 false. tooltip 离中心的水平偏移量整体由 wz_tooltip.js (设置 OffsetX) 规定, 或者利用 OFFSETX 命令设定.
例:
onmouseover="Tip('某些文字', CENTERMOUSE, true, OFFSETX, 0)"
CLICKCLOSE 当用户在 tooltip 或文件内部点击才能关闭此 tooltip. 取值: true, false.
CLOSEBTN 关闭按钮位于标题栏. 取值: true, false.
CLOSEBTNCOLORS 关闭按钮的颜色.
 
其取值必须是用逗号分隔的 4 个颜色值. 每个颜色放在一对单引号内. 整个数组用一对方括号括起来, 参见下例. 4 个颜色参数的含义如下 :
1. 背景色
2. 文字色
3. 当鼠标移至按钮上面时, 加亮的背景色
4. 当鼠标移至按钮上面时, 加亮的文字色
可以用空串 '' 作为颜色参数, 此时分别使用标题栏的背景色或文字色作为按钮的颜色.
 
例:
Tip('文字', CLOSEBTN, true, CLOSEBTNCOLORS, ['', '#66ff66', 'white', '#00cc00'], STICKY, true)
这个例子里的背景色是空串, 因此关闭按钮继承标题栏的背景色.
CLOSEBTNTEXT 关闭按钮的文字. 取值必须放在单引号里面. 例:
Tip('Tooltip 文字', CLOSEBTN, true, CLOSEBTNTEXT, '点击这里', STICKY, true)
在 wz_tooltip.js 里被整体预置为 '&nbsp;X&nbsp;' - 空格 '&nbsp;' 用来作为按钮的水平铺垫.
COPYCONTENT
这个 DIV 实际上已被放入文件的 HTML, 但它暂时转换成一个 tooltip.
COPYCONTENT 仅当使用 TagToTip() 创建 tooltip 时才有效, 也就是说, 当一个 HTML 元素被转换成 tooltip 时.
取值: true, false.
若是 true (这是 wz_tooltip.js 里预置的默认值), HTML 的仅仅一段文字(或内部 HTML)被插入 tooltip. 若是 false, 整个 HTML 元素(它的 DOM 结点)被暂时转换成一个 tooltip， 这在以下情形可能有用:
1) 如果转换成 tooltip 的 HTML 元素包含输入表格, 那么即使 tooltip 没有显示, 当前用户仍能输入.
2) tooltip 继承 HTML 元素的格式性质.
下例显示通过运用 COPYCONTENT 并设值为 false, 如何把一个 HTML 元素全部转换成 tooltip:
TagToTip('某个识别符id', COPYCONTENT, false, BORDERWIDTH, 0, PADDING, 0)
这个例子关闭了 tooltip 的自然边框 (BORDERWIDTH, 0), 并把 tooltip 的自然铺垫设成零, 这样就只显示 HTML 元素本身设定的边框及铺垫.
DELAY 经一段延迟后才显示 tooltip, 以毫秒为单位.
取值: 非负整数. 0 意为无延迟. 在 wz_tooltip.js 预置为 400.
例:
onmouseover="Tip('某些文字', DELAY, 1000)"
DURATION 提示显示的时间, 以毫秒为单位. 即使使用了 STICKY 命令或甚至仍然位于触发 tooltip 的 HTML 元素上, 本命令仍然起作用.
取值: 非负整数. 0 表示无限制 (这是默认值)
FADEIN 渐显. 其取值 (非负整数) 规定了渐显持续时间, 以毫秒为单位. 0 表示无渐变.
 
早于 v.9.0 的 Opera, Safari 的旧版本, 某些版本的 Konqueror 不支持此命令. 它们只显示通常的无渐变的提示.
FADEOUT 渐淡. 其取值 (非负整数) 规定了渐淡持续时间, 以毫秒为单位. 0 表示无渐变.
 
推荐: 与 FADEIN 联用.
 
早于 v.9.0 的 Opera, Safari 的旧版本, 某些版本的 Konqueror 不支持此命令. 它们只显示通常的无渐变的提示.
FIX 在坐标 [x, y] 的地方显示 tooltip. 取值: 用方括号括起来的一对整数. 例:
onmouseover="Tip('某些文字', FIX, [230, 874])"
你也可以调用其它地方定义的函数动态地计算坐标:
onmouseover="Tip('文字', FIX, [CalcFixX(), CalcFixY()], BORDERWIDTH, 2)"
或
onmouseover="Tip('文字', FIX, CalcFixXY(), ABOVE, true)"
在后一情形, 函数 CalcFixXY() 返回包含两个数的数组.
FOLLOWMOUSE tooltip 跟随鼠标移动. 取值: true, false. 默认: true.
 
当取值 false 关掉这个命令时, tooltip 不会跟随鼠标移动.
FONTCOLOR 改变文字色彩. 取值: 包含在单引号里面的 HTML 色彩, 例如 '#990099'
FONTFACE 改变字体.
取值: 如同 HTML 或 CSS 里使用的字体名称, 用单引号括起来, 例如 Tip('某些文字', FONTFACE, 'Arial, Helvetica, sans-serif', FONTSIZE, '12pt')
FONTSIZE 字体大小. 取值: 带单位的度量, 用单引号括起来. 单位可以是 'px', 'pt', 'em' 等.
FONTWEIGHT 字体粗细. 取值: 'normal' 或 'bold', 用单引号括起来.
LEFT 把 tooltip 放在鼠标的左边. 取值: true, false.
LEFT 和 ABOVE 联合使用.
例:
onmouseover="Tip('某些文字', LEFT, true, ABOVE, true)"
OFFSETX 从鼠标点的水平偏移. 取值: 任意整数. 也可取负值.
OFFSETY 从鼠标点的水平偏移. 取值: 任意整数. 也可取负值.
OPACITY 透明程度. 取值: 从 0 (完全透明) 到 100 (不透明) 间的整数.
 
不透明度(opacity) = 100 - 透明度 (取百分比).
另一例子 用到了 SHADOW, TEXTALIGN, BGIMG 以及 FADEIN / FADEOUT 命令.
 
早于 v.9.0 的 Opera, Safari 的旧版本, 某些版本的 Konqueror 不支持此命令.
PADDING 在边界与提示内容间的内部铺垫空间. 取值: 非负整数.
SHADOW 带有阴影的 tooltip. 取值: true, false
SHADOWCOLOR 阴影颜色. 取值: 放在单引号内部的 HTML 色彩.
例:
onmouseover="Tip('某些文字', SHADOW, true, SHADOWCOLOR, '#dd99aa')"
SHADOWWIDTH 阴影宽度 (偏移). 取值: 非负整数.
例:
onmouseover="Tip('某些文字', SHADOW, true, SHADOWWIDTH, 7)"
STICKY Tooltip 停留在初始位置直至弹出另一个 tooltip. 取值: true, false.
 
利用 DURATION 你可以让 tooltip 过一段时间后自动消失.
TEXTALIGN 规定提示内容的对齐方式. 取值: 'right', 'center', 'justify' 或 'left'.
例:
onmouseover="Tip('某些文字', TEXTALIGN, 'right')"
TITLE 显示标题栏. 取值: 放在单引号里面的标题文本. 可以包含 HTML, 它使你在设计标题栏时有更多的自由度.
TITLEALIGN 标题栏内文字的对齐方式. 取值: 'right', 'center', 'justify' 或 'left'
TITLEBGCOLOR 标题栏的背景色. 取值: 放在单引号里面的 HTML 颜色. 如果它取空串 '', 则使用边界的颜色 (参见 BORDERCOLOR 命令), 这也是默认值.
TITLEFONTCOLOR 标题文字的颜色. 取值: 放在单引号里面的 HTML 颜色. 如果它取空串 '', 则使用 tooltip 主体的背景色 (参见 BGCOLOR 命令), 这也是默认值.
TITLEFONTFACE 标题字体. 取值: 放在单引号里面的 HTML 或 CSS 格式. 如果它取空串 '', 则使用 tooltip 主体的字体, 这也是默认值.
例:
onmouseover="Tip('某些文字', TITLE, '某个标题', TITLEFONTFACE, 'Verdana,sans-serif')"
TITLEFONTSIZE 标题字体大小. 取值: 放在单引号里面的带单位的度量. 单位可取 'px', 'pt', 'em' 等. 如果它取空串 '', 则使用 tooltip 主体的字体大小.
WIDTH 宽度. 取值: 非负整数. 若取默认值 0, 宽度将根据 tooltip 的内容自动调整.
 
注意 tooltip 遵从 W3C box 模式, 也就是被指定的宽度仅应用于 tooltip 的实际内容, 不包含铺垫 (见 PADDING 命令), 边界及阴影.
